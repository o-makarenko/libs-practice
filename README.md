# 👋🏻 Hey!

Цей репозиторій містить файли-шаблони, необхідні для практичних занять з модуля "Базова робота з бібліотеками". Для успішного виконнання:

* Зробіть `Fork` цього репозиторію
* Клонуйте його на свій пристрій
* Створіть нову гілку "libs-practice"
* Робіть коміти під час роботи
* Зробіть `Push`
* Створіть `Pull Request` до оригінального репозиторію, який підпишіть своїм іменем та прізвищем
 
> Також переконайтеся, що маєте встановлені бібліотеки `numpy`, `pandas`, `matplotlib` та `kagglehub`
> В іншому разі встановіть їх через:
>
> * `pip install <library name`
> * Або підказку в PyCharm, якщо напишете `import <library name>`
